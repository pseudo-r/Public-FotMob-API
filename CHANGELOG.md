# Changelog

All notable changes to the Public-FotMob-API reverse-engineering project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to Semantic Versioning as much as an undocumented API documentation project can.

## [1.0.0] - 2026-03-26

### Added
- Comprehensive discovery and documentation of unversioned `www.fotmob.com/api` endpoints.
- Verfied endpoints: `matches`, `matchDetails`, `leagues`, `teams`, `playerData`, `searchData`, `worldnews`.
- Documented `images.fotmob.com` asset pipelines.
- Implemented Django integrated client interface inside `espn_service`.
- Finalized initial README structure mirroring the legacy ESPN API docs.
- Added `CONTRIBUTING.md` and `SPONSOR.md` for community engagement.
